# Task 2: The Creative Visionary 

**Areeba Shehzadi** | AI Researcher 

## Overview
This repository contains my work for the second task of the Generative AI Internship at DecodeLabs. The goal was to develop a complete visual identity for a tech startup using advanced AI image generation techniques.

## Project Details
*   **Focus Areas:** Advanced prompt engineering, character consistency (using image-to-image translation), aspect ratio management, and applying complex visual styles (like Cyberpunk/Neon lighting).
*   **Tools Used:** Leonardo.ai

## Assets Generated
*   `Logo_Concept` - The primary brand mark.
*   `Hero_Image` - A highly detailed, cinematic website hero image (16:9).
*   `Icon_Set` - A consistent set of UI icons.
*   `Base_Character` - The initial reference character design.
*   `Working_Character` - The same character placed in a new environment while maintaining strict visual consistency.
*   `Task_02_Creative_Visionary` - A complete log of all the exact prompts, negative prompts, and parameters used to generate these assets.
